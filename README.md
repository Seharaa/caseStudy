## Description

Simulates a customer service call center using **SimPy**. Compares three scenarios:

1. Baseline: 5 agents, 10 calls/hour
2. More Agents: 7 agents, 10 calls/hour
3. Higher Load: 5 agents, 12 calls/hour

## Outputs

* Average customer wait time
* Agent utilization
* Queue length
* Customers served

## Usage

1. Install SimPy: pip install simpy
2. Run the simulation:
    python CallCenterSimultion.py
